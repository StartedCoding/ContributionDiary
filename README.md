# 🧠 ContributionDiary

> A personal log of my open-source contributions, what I learned, and how I solved challenges.

---

## 📖 Purpose
This repo documents my journey in open source — every PR, bug fix, feature, or documentation update I make.

Each entry reflects:
- 🧩 What I contributed
- 🧠 What I learned
- ⚙️ How I solved problems
- 🪶 Tags and reflections

---

## 📅 How It’s Organized
Each contribution is logged as a Markdown file inside the `/entries` folder.

**Format:**  
`YYYY-MM-DD-project-name.md`
